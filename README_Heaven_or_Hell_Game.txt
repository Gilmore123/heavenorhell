
# Heaven or Hell Game - Instructions

Thank you for purchasing the Heaven or Hell game! This README file will guide you through the process of setting up and playing the game.

## 1. What You Will Receive
You have downloaded a .zip file named `Heaven_or_Hell_Game.zip`, which contains the following:
- `RunGame.html` (Main game file- click to play)
- `gamemanual.pdf (Contains game instructions)
- `style.css` (CSS file for styling)
- `script.js` (JavaScript file for game functionality)
- Any additional assets such as images or sound files (if included)

## 2. How to Set Up and Play the Game
Follow these steps to get started with the game:

1. **Extract the .zip file:**
   - Right-click on the `Heaven_or_Hell_Game.zip` file and select "Extract All" (Windows) or "Open With > Archive Utility" (Mac).
   - Choose a location on your computer where you want to extract the files.

2. **Open the Game:**
   - Once the files are extracted, locate the `RunGame.html` file in the extracted folder.
   - Double-click on gamemanual.pdf to read game instructions.
   - Double-click on `RunGame.html` to open it in your default web browser.
   - The game will launch, and you can start playing immediately!
 
## 3. System Requirements
- A modern web browser such as Chrome, Firefox, Safari, or Edge.
- No internet connection is required to play the game after download.

## 4. Mobile Compatibility
- The game is designed to work on both desktop and mobile web browsers.
- To play on mobile, open the `RunGame.html` file using a mobile web browser (Chrome, Safari, etc.).
- Some game features or layouts may be better suited to larger screens.

## 5. Troubleshooting
If you encounter any issues:
- Make sure all files were extracted correctly from the .zip file.
- Verify that you are opening the `RunGame.html` file in a supported web browser.
- Try clearing your browser's cache if the game is not loading properly.
